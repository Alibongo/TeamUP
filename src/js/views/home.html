about

team:up was created by byte:me, a team of developers based in London. The inspiration for team:up began as a result of a single question: what if an agency model could be replicated in an online network, with no overheads and greater freedom and flexibility for teams? team:up is a networking tool which allows you to access a network and build a team to work on freelance projects, track team members and budget resource allocation.

How it works

Step 1: Register and Login to create a profile listing the roles you would be looking for.

Step 2: If you are looking for work, you can search for Projects by role and apply to join projects.

Step 3: Create your project which will become searchable on the database. Your project page will allow you to track and accept applicants in order to build your perfect team.
